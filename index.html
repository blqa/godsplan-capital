Núcleo del Sistema

Cuentas Espejo

Crea múltiples bóvedas virtuales que representan tu realidad (Débito BBVA, Efectivo Cartera, Tarjeta de Crédito).

Registro de Transacciones

Sistema ultrarrápido para ingresar ingresos, gastos y transferencias internas entre tus propias cuentas.

Categorización y Etiquetas

Asigna gastos a categorías fijas (Renta, Comida) y utiliza etiquetas dinámicas (#ViajeCancun2026) para proyectos.

Presupuestos Inteligentes

Define límites mensuales por categoría. Recibe alertas proactivas antes de pasarte de tu presupuesto.

Dashboard Financiero

Pantalla principal con gráficos visuales que calculan tu patrimonio neto (activos - pasivos), flujo de caja y progreso de metas.

Flujos de Alimentación de Datos

100% Manual

Modo Control Total

Tú registras cada café, pago y nómina a mano. Es el método más seguro y genera la mayor consciencia psicológica de gasto.

Semi-Automatizado

Importación

Descargas el estado de cuenta en Excel/CSV de BBVA y la app tiene un importador que acomoda los gastos automáticamente.

Automatizado

Vía API (Avanzado)

Uso de servicios de "Open Banking" para conectarse de forma segura a tu banco y jalar movimientos en tiempo real (solo lectura).

Seguridad Inquebrantable

Aunque no haya dinero real, tus datos financieros son extremadamente sensibles. La teoría exige que se cumplan estas 3 reglas:

Base de datos encriptada

Protección criptográfica para tu información almacenada.

Privacidad Absoluta

Nadie más que tú tiene acceso a ver en qué gastas. Sin recolección de datos para terceros.

Autenticación (2FA)

El inicio de sesión cuenta con las medidas de seguridad más estrictas de autenticación.
